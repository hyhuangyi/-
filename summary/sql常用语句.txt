								 常用sql语句

1.增        INSERT INTO SYS_TEAM_USER (TEAM_USER_ID,TEAM_ID,USER_ID) VALUES (20000,10000,200);

2.删        DELETE FROM SYS_TEAM_USER WHERE TEAM_USER_ID=20000;

3.改        UPDATE SYS_TEAM_USER SET CREATE_USER=2,UPDATE_USER=2 WHERE TEAM_USER_ID=10178;

4.查        SELECT *FROM SYS_TEAM_USER;

5.序列创建  CREATE SEQUENCE HY  AS INTEGER  START WITH 1 INCREMENT BY 1 MINVALUE 1 MAXVALUE 200000000 CACHE 20 NO CYCLE  NO ORDER;

 删除序列   DROP SEQUENCE HY; 
  
 取下一个值 SELECT NEXTVAL(NEXT VALUE) FOR HY FROM SYSIBM.SYSDUMMY1;（内存特殊表 伪表）

 取当前值   SELECT PREVVAL(PREVIOUS VALUE) FOR HY FROM SYSIBM.SYSDUMMY1;

 修改序列开始值  ALTER SEQUENCE HY RESTART WITH 10000;

 查询所有序列     SELECT *FROM SYSCAT.SEQUENCES

6.系统当前日期  SELECT CURRENT DATE FROM SYSIBM.SYSDUMMY1;

7.查询表结构    SELECT COLNAME,TYPENAME,LENGTH,'FALSE','FALSE','FALSE','',REMARKS  FROM SYSCAT.COLUMNS WHERE TABNAME='LN_LOAN_INFO'

8.查询表      SELECT TABNAME FROM SYSCAT.TABLES WHERE  TABSCHEMA='DB2ADMIN'

9.db2分页     SELECT *FROM(SELECT A.*,ROW_NUMBER() OVER(ORDER BY LOAN_INFO_ID) AS ROWID FROM(SELECT *FROM LN_LOAN_INFO)A) WHERE ROWID BETWEEN 1 AND 5

	      SELECT *FROM(SELECT A.*,ROW_NUMBER() OVER(ORDER BY LOAN_INFO_ID)AS ROWID FROM LN_LOAN_INFO AS A)B WHERE B.ROWID<10 

10.通配符转义 SELECT *FROM S_USER  where  S_USER.ENNAME like '%/%%'  ESCAPE '/' 如果想要查带通配字符的字符串，则通配字符需要转义  (查询带%的字符串)

11.mybatis中$和#    #会预编译，而$不会，通常在字段设置值的时候，我们要使用#，假如我们的sql表名是一个变量tab_name 则用$
