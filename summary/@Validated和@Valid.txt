								     @Validated 和@Valid

@Validated 和@Valid  配合@ModelAttribute 与BindingResult 对对象进行校验 ，如果校验不通过，那么错误信息就会封装到BindingResult对象了，可以通过bindingResult的相关方法获
取详细的错误信息并返回给用户。如果不加BindingResult则会抛出异常。


    @RequestMapping("exceptionTest")
    @ResponseBody
    public String exceptionTest( @Valid@ModelAttribute City city,BindingResult result) throws Exception{
        if(result.hasErrors()){
            throw new BindException(result);
        }
        return city.getCode();
    }
 


@Validated 配合MethodValidationPostProcessor拦截器 实现对对个参数进行校验,类Controller前加@Validated

    @RequestMapping("exceptionTest1")
    @ResponseBody
    public String exceptionTest1(@NotEmpty@RequestParam String name)  {

        return name;
    }
